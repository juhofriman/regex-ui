Cheatsheet

Boolean OR: foo|bar
Grouping: potat(a|o)
Quantification
?: zero or one
*: zero or more
.*?a: zero or more (lazy, anything until first a)
+: one or more
{n}: match n times
{n,}: match n or more times
{n,m}: match n to m times
Brackets:
[abc] matches a, b and c
[a-z] lower case letters
[^abc] any character but not a, b and c
Special characters:
. matches any character
^ matches beginning of line
$ matches end of line
Character classes
\w: [A-Za-z0-9_]
\W: [^A-Za-z0-9_]
\d: digits
\D: non digits
\s: whitespaces
\S: non whitespaces
